# Teste Strong Way Este é um projeto WordPress que inclui um tema personalizado e um plugin para gerenciar informações sobre pássaros. ## Instalação 1. Clone este repositório para o diretório `wp-content/themes/` do seu site WordPress. 2. Ative o tema "Meu Tema Passarinho" na área de administração do WordPress. 3. Faça o upload e ative o plugin "Meu Plugin de Pássaros" na área de administração do WordPress. ## Tema Personalizado O tema personalizado inclui um layout responsivo para exibir informações sobre pássaros. Ele também possui funcionalidades para exibir uma lista de pássaros e detalhes individuais de cada pássaro. ## Plugin de Pássaros O plugin de pássaros adiciona um novo tipo de postagem chamado "Pássaros" e uma taxonomia chamada "Região". Ele também inclui um modelo para exibir os detalhes de um pássaro individual. ## Contribuindo Sinta-se à vontade para abrir uma issue ou enviar um pull request para contribuir com melhorias neste projeto. ## Licença Este projeto está sob a licença [MIT License](LICENSE).
